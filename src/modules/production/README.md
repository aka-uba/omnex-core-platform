# Production & Product Module

Comprehensive production and product management module with product management, BOM (Bill of Materials), production orders, and stock movement tracking.

## Features

- **Product Management**: Complete product catalog with variants, attributes, and specifications
- **Multi-level BOM**: Support for complex bill of materials with multiple levels
- **Production Planning**: Production scheduling and capacity planning
- **Work Center Management**: Manage production work centers and resources
- **Routing Configuration**: Define production routes and operations
- **Quality Control**: Quality checks and inspection management
- **Batch & Serial Tracking**: Track products by batch and serial numbers
- **Production Orders**: Create and manage production orders
- **Stock Management**: Real-time stock tracking and movement management
- **Production Analytics**: Detailed production reports and analytics

## Dependencies

- **Locations Module** (required): For location-based production and inventory management

## Installation

The module is automatically available when installed. Ensure the Locations module is installed and activated first.

## Configuration

Configure the module through the module settings page after activation.

## Usage

1. Navigate to Production from the main menu
2. Access Dashboard, Products, BOM, Production Orders, or Stock submenus
3. Manage production operations through the respective interfaces

## Permissions

The module includes the following permissions:
- `production.product.create` - Create products
- `production.product.update` - Update products
- `production.product.delete` - Delete products
- `production.order.create` - Create production orders
- `production.order.update` - Update production orders
- `production.stock.manage` - Manage stock movements
