# Human Resources Module

Comprehensive human resources management module with employee management, leave tracking, attendance monitoring, and payroll processing.

## Features

- **Employee Management**: Complete employee profiles, organizational structure, and employee lifecycle management
- **Leave Management**: Leave request tracking, approval workflows, and leave balance management
- **Attendance Monitoring**: Time tracking, attendance records, and absence management
- **Payroll Processing**: Automated payroll calculations, salary management, and payment processing
- **Performance Reviews**: Employee performance evaluation and review management
- **Reporting**: Comprehensive HR reports and analytics

## Dependencies

- **Locations Module** (required): For location-based employee assignments and organizational structure

## Installation

The module is automatically available when installed. Ensure the Locations module is installed and activated first.

## Configuration

Configure the module through the module settings page after activation.

## Usage

1. Navigate to Human Resources from the main menu
2. Access Employees, Leaves, or Payrolls submenus
3. Manage your HR operations through the respective interfaces

## Permissions

The module includes the following permissions:
- `hr:employees:read` - View employees
- `hr:employees:write` - Create and edit employees
- `hr:employees:delete` - Delete employees
- `hr:leaves:read` - View leaves
- `hr:leaves:write` - Create and edit leaves
- `hr:leaves:delete` - Delete leaves
- `hr:payrolls:read` - View payrolls
- `hr:payrolls:write` - Create and edit payrolls
- `hr:payrolls:delete` - Delete payrolls
