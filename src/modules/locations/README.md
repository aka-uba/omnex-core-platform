# Locations Module

Core location management module for managing locations, hierarchies, and geographic data.

## Features

- Location management with hierarchical structure
- Location hierarchy visualization and management
- Geographic data support
- Location-based filtering and reporting
- Integration with other business modules

## Dependencies

This is a core module with no dependencies. Other modules may depend on this module.

## Usage

The Locations module provides core location functionality that is required by other business modules such as:
- Accounting
- Human Resources
- Maintenance
- Production & Product

## Configuration

No special configuration required. The module is automatically available when installed.
